
NAME
    ceph tempurl - Generate temporary HTTP URL.

SYNOPSIS
    ceph tempurl --name <object-name> --connection <pathname> [--container <container-name>]

OPTIONS
    --name <object-name>
    Object name used in CEPH storage.

    --connection | -c <pathname>
    CEPH storage connection configuration file.

    --container <container-name>
    The container name on CEPH storage.
